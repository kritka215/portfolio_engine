A premium, AI-powered investment portfolio dashboard built with **Python**, **Gradio**, and **Plotly**.

## Features
- **Interactive Dashboard**: Real-time updates based on risk profile and investment goals.
- **Visual Analytics**:
  - Asset Allocation Donut Charts
  - Wealth Projection Area Charts (Monte Carlo Simulation)
  - Key Metrics (Sharpe Ratio, Volatility, ROI)
- **Responsive Design**: Custom CSS and soft theme for a professional look.

## Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd portfolio-project
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   python app.py
   ```

## Tech Stack
- **Frontend**: Gradio (Blocks API)
- **Visualizations**: Plotly Express & Graph Objects
- **Data**: Pandas
- **Backend Logic**: Custom Risk & Portfolio Engines
